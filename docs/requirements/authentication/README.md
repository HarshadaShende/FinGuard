# Purpose

The Authentication module provides secure access to the FinGuard Digital Banking Platform.

It is responsible for verifying user credentials, issuing JWT access tokens, protecting secured APIs and managing authenticated user sessions.

---

# Implemented Features

- Customer Login
- JWT Authentication
- Protected Profile API
- Logout
- Authentication Middleware
- Request Validation

---

# Planned Features

- Refresh Token
- Forgot Password
- Reset Password
- Account Lock after Failed Attempts
- Multi-Factor Authentication (MFA)
